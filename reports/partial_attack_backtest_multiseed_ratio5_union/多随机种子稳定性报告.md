# 部分股票白盒攻击回测多随机种子稳定性报告

## 1. 实验目的

在单次固定随机种子 `seed=0` 的部分股票攻击回测已经观察到明确退化之后，本轮实验进一步检验该结论是否对随机种子稳定。核心问题是：在“完整测试区间、每日随机攻击 `5%` 股票、攻击目标为最大化 MSE”的口径下，FGSM 与 PGD 对组合层面的退化方向和幅度，是否会随着随机种子的变化而发生根本性反转。

## 2. 方法修正与实验设置

本轮复跑沿用与单次实验完全一致的攻击和回测配置，唯一变化是将随机种子扩展为 `seed=0,1,2,3,4`。需要特别说明的是，最初直接复用 `seed=0` 对应的 partial asset 去跑 `seed>0` 时，出现了 `attackable_count` 异常下降的问题；根因并不是攻击本身不稳定，而是该资产目录仅覆盖 `seed=0` 选中的 key。为避免这一偏差，本轮先对 `2025-01-01` 至 `2025-10-31` 区间内 `seeds 0..4` 的候选攻击 key 取并集，再构建一份共享原始窗口资产。

共享资产的构造结果如下：

- 候选 key 并集规模：`237088`
- `pred.pkl` 与 `label.pkl` 的有效交集：`236554`
- 成功重建的原始 `OHLCV` 窗口数：`230077`
- 缺失 raw window 的 key 数：`6477`

本轮共享资产仅保留当前多 seed 复跑真正需要的三项内容，即 `matched_reference.csv`、`normalization_stats.json` 和 `matched_ohlcv_windows.pt`，不再重复导出 `matched_feature_windows.pt`。由于本轮稳定性检验关注的是攻击后组合退化方向是否一致，而不是再次收紧 feature-level clean gate，因此这一最小资产口径是足够的。

## 3. 各随机种子的覆盖率

五个随机种子的有效覆盖情况如下表所示。

| seed | selected_available_count | attackable_count |
| ---: | ---: | ---: |
| 0 | 50874 | 50874 |
| 1 | 50892 | 50892 |
| 2 | 50888 | 50888 |
| 3 | 50846 | 50846 |
| 4 | 50875 | 50875 |

从统计量看，`attackable_count` 的均值为 `50875.0`，标准差仅为 `18.03`，最小值和最大值分别为 `50846` 与 `50892`。这说明，在共享资产修正之后，多随机种子实验的输入覆盖率已经高度稳定，不再存在由资产缺口主导的实验偏差。

## 4. 主结果

本轮仍以 `partial_clean` 作为主对照。首先考察辅助对照 `partial_clean - reference_clean`，其作用是衡量“仅替换被攻击子集的 clean 前向分数”本身会给组合回测带来多大偏移。五个种子上，该差值在年化超额收益(含费)上的均值为 `-0.01752`，标准差为 `0.02789`，取值范围为 `[-0.05130, 0.01195]`；在 `rank_ic_mean` 上的均值仅为 `0.000167`，标准差为 `0.000165`。这表明，`partial_clean` 相对 `reference_clean` 的收益侧偏移存在一定波动，但排序侧偏移非常小。因此，后续攻击效果仍应主要以 `partial_fgsm - partial_clean` 与 `partial_pgd - partial_clean` 为准，而不应直接与 `reference_clean` 比较。

真正的稳定性判断来自攻击组相对 `partial_clean` 的差值。五个种子上的统计结果如下：

| 差值组别 | 年化超额收益(含费) | 最大回撤(含费) | RankIC 均值 | 信息比率(含费) |
| --- | ---: | ---: | ---: | ---: |
| `partial_fgsm - partial_clean` 平均值 | `-0.25310` | `-0.04930` | `-0.01310` | `-1.77472` |
| `partial_fgsm - partial_clean` 标准差 | `0.04631` | `0.00461` | `0.00015` | `0.35473` |
| `partial_pgd - partial_clean` 平均值 | `-0.36236` | `-0.05440` | `-0.01451` | `-2.69830` |
| `partial_pgd - partial_clean` 标准差 | `0.03500` | `0.00398` | `0.00019` | `0.29081` |

更重要的是，以上退化方向在五个种子上完全一致。对于 FGSM，`partial_fgsm - partial_clean` 的年化超额收益、最大回撤、`rank_ic_mean` 和信息比率四项差值在 `5/5` 个种子上全部为负；对于 PGD，同样四项差值也在 `5/5` 个种子上全部为负。这说明，一旦把比较基线固定在 `partial_clean`，局部攻击对组合表现的削弱并不是单个种子的偶然现象，而是跨种子的稳定结果。

## 5. FGSM 与 PGD 的相对强弱

从均值上看，PGD 在四个主指标上都比 FGSM 更强。相较于 `partial_clean`，PGD 使年化超额收益的平均降幅达到 `-0.36236`，明显大于 FGSM 的 `-0.25310`；`rank_ic_mean` 的平均降幅为 `-0.01451`，也大于 FGSM 的 `-0.01310`；信息比率的平均降幅为 `-2.69830`，同样大于 FGSM 的 `-1.77472`。在这些指标上，PGD 的退化幅度在 `5/5` 个种子上都比 FGSM 更大。

最大回撤的结论略弱一些。虽然 PGD 的平均回撤恶化幅度 `-0.05440` 仍大于 FGSM 的 `-0.04930`，但从逐 seed 比较看，PGD 在 `3/5` 个种子上的回撤恶化幅度更大，而不是 `5/5` 全胜。这说明，在当前口径下，PGD 对收益和排序能力的打击具有更稳定的优势，而对最大回撤的优势虽然在均值上成立，但个别种子上会被组合路径噪声部分掩盖。

## 6. 结论

本轮多随机种子复跑给出三个较稳固的结论。第一，共享资产修正后，多 seed 实验的输入覆盖率已经稳定，`attackable_count` 基本固定在 `5.09` 万量级，因此后续结果不再受资产覆盖偏差主导。第二，以 `partial_clean` 为主对照时，FGSM 与 PGD 在五个随机种子上都稳定导致组合退化，且年化超额收益、最大回撤、`rank_ic_mean` 与信息比率四项指标的退化方向全部一致。第三，PGD 在收益、排序和信息比率上的破坏效应稳定强于 FGSM；对最大回撤而言，PGD 的平均退化更大，但优势不如前述指标那样绝对。

因此，就“每日随机攻击 `5%` 股票”的当前设定而言，可以认为组合级退化已经具备跨随机种子的稳定性证据。下一步若继续增强证据强度，最自然的方向是扩展攻击比例维度，例如比较 `1% / 5% / 10%` 的退化曲线，或在补齐 `matched_feature_windows.pt` 后加入更严格的 feature-level clean gate 复核。
