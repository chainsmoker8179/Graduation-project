# LSTM 单模型攻击证据补强报告

## 1. 主结果回顾
- 在 `5%` 每日局部攻击、`seed=0..4` 的主口径下，FGSM 相对 `partial_clean` 的年化超额收益均值退化为 `-0.2531`，`RankIC` 均值退化为 `-0.013098`。
- 在相同口径下，PGD 相对 `partial_clean` 的年化超额收益均值退化为 `-0.3624`，`RankIC` 均值退化为 `-0.014508`。
- 这些结果与已有多随机种子回测结论一致，说明当前主文仍应以 `5%` 多 seed 作为核心证据。

## 2. 统计显著性
- `partial_fgsm_vs_partial_clean` 在 `超额收益` 上的日度平均差值为 `-0.001177`，`p=9.72e-03`。
- `partial_pgd_vs_partial_clean` 在 `超额收益` 上的日度平均差值为 `-0.001674`，`p=2.29e-03`。
- `partial_fgsm_vs_partial_clean` 在 `RankIC` 上的日度平均差值为 `-0.012988`，`p=1.46e-34`。
- `partial_pgd_vs_partial_clean` 在 `RankIC` 上的日度平均差值为 `-0.014378`，`p=1.44e-34`。
- `partial_fgsm_vs_partial_clean` 在 `年化超额收益` 上的 block bootstrap 区间为 `[-0.5354, -0.0701]`。
- `partial_pgd_vs_partial_clean` 在 `年化超额收益` 上的 block bootstrap 区间为 `[-0.7454, -0.1556]`。

## 3. 排序机制
- `FGSM` 的 `Top-K 重合率` 平均值为 `0.1413`。
- `PGD` 的 `Top-K 重合率` 平均值为 `0.0408`。
- `FGSM` 的 `分数 Spearman` 平均值为 `0.9728`。
- `PGD` 的 `分数 Spearman` 平均值为 `0.9690`。
- `FGSM` 的 `平均绝对排名位移` 平均值为 `124.2376`。
- `PGD` 的 `平均绝对排名位移` 平均值为 `134.1667`。
- 这说明攻击并非仅在少量样本上抬高损失，而是会系统性扰动横截面排序，从而进一步传导到组合构建层面。
