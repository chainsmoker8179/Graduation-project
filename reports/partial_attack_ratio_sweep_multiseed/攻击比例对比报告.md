# 部分股票攻击比例对比报告

## 1. 实验目的

本轮实验在已经完成 `5%` 攻击比例、多随机种子稳定性验证之后，进一步回答一个更具体的问题：当每日被攻击股票的覆盖率从 `1%` 提升到 `5%`，再提升到 `10%` 时，组合级回测指标是否会出现更强退化。

本报告固定如下口径：

- 模型：legacy LSTM
- 测试区间：`2025-01-01` 至 `2025-10-31`
- 随机种子：`seed=0..4`
- 攻击比例：`1% / 5% / 10%`
- 攻击目标：最大化 MSE
- 攻击器：FGSM、PGD
- 主对照：`partial_clean`

也就是说，本报告的核心比较对象是：

- `partial_fgsm - partial_clean`
- `partial_pgd - partial_clean`

而不是直接把攻击结果与 `reference_clean` 对比。这样可以更好地隔离“局部替换 clean 分数本身带来的残余偏差”。

## 2. 共享资产策略

比例对比不能直接复用某一个固定 `seed` 或固定 `ratio` 的 partial asset。原因是不同配置下能够被攻击到的资产覆盖集并不完全相同，如果直接复用旧资产，会把“样本覆盖差异”错误地混入“攻击比例差异”。

因此，本轮采用共享资产策略：

1. 对 `ratio=1% / 5% / 10%` 与 `seed=0..4` 分别生成候选攻击 key。
2. 对全部候选 key 取并集。
3. 仅为这个总并集导出一份共享 raw-only 资产。
4. 在三档比例的全部实验中统一复用这份资产。

共享资产规模如下：

| 项目 | 数值 |
| --- | ---: |
| `1%` 候选 key 并集 | `51,923` |
| `5%` 候选 key 并集 | `237,088` |
| `10%` 候选 key 并集 | `428,802` |
| 三档比例总并集 | `591,791` |
| `matched_reference_rows` | `590,504` |
| `exported_sample_rows` | `574,386` |
| `missing_raw_keys` | `16,118` |

这意味着三档比例的实验都建立在同一份原始输入资产覆盖集合上，比例差异主要来自攻击掩码本身，而不是来自“可攻击样本是否存在”的偏差。

## 3. 样本覆盖与可攻击规模

五个随机种子下，各比例的 `attackable_count` 非常稳定：

| 比例 | 候选 key 并集 | `attackable_count` 均值 | 标准差 |
| --- | ---: | ---: | ---: |
| `1%` | `51,923` | `10,281.0` | `17.35` |
| `5%` | `237,088` | `50,875.0` | `18.03` |
| `10%` | `428,802` | `101,670.8` | `34.65` |

这里有两个直接结论：

1. 多随机种子下的可攻击样本数波动很小，说明共享资产策略有效压低了覆盖偏差。
2. `attackable_count` 随攻击比例近似线性上升，因此后续比较三档比例的组合退化具有可解释性。

## 4. 攻击退化结果

### 4.1 FGSM 相对 `partial_clean` 的退化

| 指标 | `1%` | `5%` | `10%` |
| --- | ---: | ---: | ---: |
| 年化超额收益率（含成本） | `-0.2026 ± 0.0374` | `-0.2531 ± 0.0463` | `-0.3313 ± 0.0125` |
| 最大回撤（含成本） | `-0.0426 ± 0.0138` | `-0.0493 ± 0.0046` | `-0.0500 ± 0.0104` |
| `RankIC` 均值 | `-0.002761 ± 0.000124` | `-0.013098 ± 0.000146` | `-0.025438 ± 0.000473` |
| 信息比率（含成本） | `-1.3538 ± 0.3213` | `-1.7747 ± 0.3547` | `-2.3932 ± 0.0900` |

FGSM 的四个主指标在三档比例上都满足相同趋势：比例越高，平均退化越强。并且这四个指标在五个随机种子上全部为负，方向一致性为 `5/5`。

### 4.2 PGD 相对 `partial_clean` 的退化

| 指标 | `1%` | `5%` | `10%` |
| --- | ---: | ---: | ---: |
| 年化超额收益率（含成本） | `-0.3014 ± 0.0198` | `-0.3624 ± 0.0350` | `-0.4128 ± 0.0233` |
| 最大回撤（含成本） | `-0.0438 ± 0.0162` | `-0.0544 ± 0.0040` | `-0.0661 ± 0.0141` |
| `RankIC` 均值 | `-0.003078 ± 0.000124` | `-0.014508 ± 0.000186` | `-0.028142 ± 0.000494` |
| 信息比率（含成本） | `-2.1382 ± 0.1664` | `-2.6983 ± 0.2908` | `-3.0935 ± 0.1924` |

PGD 的四个主指标同样在三档比例上全部呈现严格的均值递减趋势，并且五个随机种子上全部为负，方向一致性也是 `5/5`。

### 4.3 `partial_clean` 残余偏差

为了确认比例对比没有被 clean 拼接误差主导，需要看 `partial_clean - reference_clean` 的量级：

| 指标 | `1%` | `5%` | `10%` |
| --- | ---: | ---: | ---: |
| 年化超额收益率（含成本） | `-0.0001 ± 0.0093` | `-0.0175 ± 0.0279` | `-0.0041 ± 0.0231` |
| 最大回撤（含成本） | `-0.0004 ± 0.0037` | `0.0001 ± 0.0054` | `-0.0033 ± 0.0074` |
| `RankIC` 均值 | `0.000052 ± 0.000070` | `0.000167 ± 0.000165` | `0.000147 ± 0.000174` |
| 信息比率（含成本） | `0.0141 ± 0.0576` | `-0.0902 ± 0.2227` | `-0.0151 ± 0.1725` |

这些 clean 残余项整体上明显小于攻击退化项。以年化超额收益率为例，`10%` 比例下 clean 残余均值仅为 `-0.0041`，而 FGSM 与 PGD 的对应退化均值分别达到 `-0.3313` 和 `-0.4128`。因此，本轮比例对比的主结论仍然可以归因于攻击扰动，而不是归因于局部 clean 替换。

## 5. 趋势分析

### 5.1 比例上升是否带来更强退化

答案是肯定的，而且结论非常整齐。

对 FGSM 和 PGD 分别考察四个主指标：

- 年化超额收益率（含成本）
- 最大回撤（含成本）
- `RankIC` 均值
- 信息比率（含成本）

共 `2 x 4 = 8` 组趋势检查中，三档比例的均值全部满足：

- `1% >= 5% >= 10%`

这里的“更小”表示更差，因为我们比较的是相对 `partial_clean` 的指标差值。换句话说，攻击比例从 `1%` 扩大到 `10%` 后，无论使用 FGSM 还是 PGD，组合层面的退化都同步增强，没有出现反向或混乱趋势。

### 5.2 PGD 是否继续强于 FGSM

总体上仍然成立。

在三档比例上：

- 年化超额收益率退化：PGD 在 `15/15` 个 seed-ratio 组合上都强于 FGSM。
- `RankIC` 退化：PGD 在 `15/15` 个 seed-ratio 组合上都强于 FGSM。
- 信息比率退化：PGD 在 `15/15` 个 seed-ratio 组合上都强于 FGSM。
- 最大回撤退化：PGD 在 `1%` 和 `5%` 下为 `3/5` 个种子更强，在 `10%` 下为 `5/5` 个种子更强。

因此，PGD 相比 FGSM 的优势在收益类和排序类指标上非常稳定，在最大回撤上则表现为“总体更强，但低比例下个别种子不必然更强”。

### 5.3 哪些指标对比例变化最敏感

从均值变化幅度看，最稳定的两个趋势是：

1. `RankIC` 均值下降随比例上升而持续放大。
2. 信息比率下降随比例上升而持续放大。

例如：

- FGSM 的 `RankIC` 均值退化从 `-0.002761` 扩大到 `-0.025438`
- PGD 的 `RankIC` 均值退化从 `-0.003078` 扩大到 `-0.028142`
- FGSM 的信息比率退化从 `-1.3538` 扩大到 `-2.3932`
- PGD 的信息比率退化从 `-2.1382` 扩大到 `-3.0935`

相比之下，最大回撤虽然也整体变差，但 FGSM 在 `5% -> 10%` 之间的均值变化较小，说明回撤对比例变化的敏感性不如收益和排序指标那样线性。

## 6. 结论

本轮比例对比可以得到三个核心结论：

1. 每日仅攻击一小部分股票已经足以引起稳定的组合级退化，而把覆盖率从 `1%` 提升到 `5%`、再提升到 `10%` 会进一步放大这种退化。
2. PGD 在大多数指标上继续稳定强于 FGSM，尤其是在年化超额收益、`RankIC` 和信息比率三个维度上，优势具有跨比例、跨种子的稳健性。
3. `partial_clean - reference_clean` 的残余偏差仍显著小于攻击效应，因此本轮结论主要反映的是攻击比例变化本身，而不是 clean 拼接误差。

从论文叙述角度看，这一轮实验把此前“局部攻击会导致组合退化”的单比例结论，进一步推进为“组合退化强度随攻击覆盖率上升而系统性增强”的证据。

## 7. 配套产物

本报告对应的结构化结果文件位于同目录：

- `ratio_sweep_seed_metrics.csv`
- `ratio_sweep_summary_stats.csv`
- `ratio_sweep_trend_summary.json`

其中：

- `ratio_sweep_seed_metrics.csv` 用于保存 `15` 个 `ratio-seed` 组合的 seed 级结果。
- `ratio_sweep_summary_stats.csv` 用于保存三档比例下各指标的均值、标准差和方向一致性。
- `ratio_sweep_trend_summary.json` 用于保存共享资产规模、单调趋势检查和 PGD 对 FGSM 的相对强度统计。
