# 部分股票白盒攻击回测多随机种子稳定性报告

## 1. 实验目的

此前组合层 `5%` 多随机种子实验已经在无约束攻击口径下验证过：每日随机攻击部分股票后，`FGSM` 与 `PGD` 都会稳定导致组合表现退化。但随着样本级攻击主线已经切换到带金融物理与统计约束的 `physical_stat` 口径，当前更关键的问题变成：

在“每日随机攻击 `5%` 股票、随机种子 `0..4`”这一组合层设定下，如果把攻击器也切换到 `physical_stat`，组合退化是否仍然稳定成立，以及 `FGSM` / `PGD` 的相对强弱是否会发生变化。

## 2. 实验设置

- 攻击比例：`5%`
- 随机种子：`0, 1, 2, 3, 4`
- 模型：`LSTM`
- 攻击器：`FGSM` / `PGD`
- 约束模式：`physical_stat`
- 统计约束超参：
  - `lambda_ret = 0.8`
  - `lambda_candle = 0.4`
  - `lambda_vol = 0.3`

本轮首先重建了新的共享 raw-only 攻击资产，其统计如下：

- 候选 key 并集规模：`236960`
- `pred.pkl` 与 `label.pkl` 的有效交集：`236412`
- 成功重建的原始 `OHLCV` 窗口数：`230072`

这说明本轮 `physical_stat` 组合层实验的输入覆盖率与此前旧实验处于同一量级，不存在因为资产严重缺失而导致的覆盖偏差。

这说明即使切换到新的共享资产与新攻击口径，五个随机种子的可攻击覆盖率仍然非常稳定。

## 4. 主结果

仍以 `partial_clean` 作为主对照，考察攻击后相对 `partial_clean` 的组合退化差值。主结果如下：

| 差值组别 | 年化超额收益(含费) | 最大回撤(含费) | RankIC 均值 | 信息比率(含费) |
| --- | ---: | ---: | ---: | ---: |
| `partial_fgsm - partial_clean` 平均值 | `-0.00007413` | `-0.01843885` | `-0.01137699` | `0.05958811` |
| `partial_fgsm - partial_clean` 标准差 | `0.04262264` | `0.01461744` | `0.00019645` | `0.35981568` |
| `partial_pgd - partial_clean` 平均值 | `-0.09201836` | `-0.03226386` | `-0.01113051` | `-0.61149516` |
| `partial_pgd - partial_clean` 标准差 | `0.05728175` | `0.02338459` | `0.00018343` | `0.41596078` |

从方向一致性看：

- `partial_pgd - partial_clean`
  - 年化超额收益：`5/5` 个种子均为负
  - 最大回撤：`5/5` 个种子均为负
  - `rank_ic_mean`：`5/5` 个种子均为负

  - 年化超额收益：只有 `3/5` 个种子为负
  - 最大回撤：`5/5` 个种子均为负
  - `rank_ic_mean`：`5/5` 个种子均为负
  - 信息比率：只有 `2/5` 个种子为负

## 5. 结果解读

### 5.1 `PGD + physical_stat` 的组合层退化仍然稳定成立

`PGD` 组的结论非常明确。相较于 `partial_clean`：

- 年化超额收益均值下降 `-0.0920`
- 最大回撤均值进一步恶化 `-0.0323`
- `rank_ic_mean` 均值下降 `-0.0111`
- 信息比率均值下降 `-0.6115`

更重要的是，上述四项指标在五个随机种子上全部同向退化。也就是说，即使引入 `physical_stat` 约束后，`PGD` 依然能够稳定削弱组合表现，这一结论并没有因为攻击更保守而消失。

### 5.2 `FGSM + physical_stat` 被明显收紧

与 `PGD` 相比，`FGSM` 在 `physical_stat` 口径下被压缩得更厉害。其表现是：

- `rank_ic_mean` 仍稳定下降，均值为 `-0.01138`
- 最大回撤也稳定恶化，均值为 `-0.01844`
- 但年化超额收益均值几乎为 `0`
- 信息比率均值反而略为正值 `0.0596`

这说明在当前默认约束强度下，单步攻击难以同时兼顾“收益打击”与“约束满足”，从而使其在收益侧和风险调整收益侧的破坏能力显著弱于无约束版本。

### 5.3 `PGD` 与 `FGSM` 的相对强弱发生了结构性变化

- 在 `rank_ic_mean` 上，两者都稳定恶化，但 `FGSM` 的降幅反而略大于 `PGD`

这说明引入 `physical_stat` 约束后，`PGD` 仍然是更强的组合层攻击器，但“更强”主要体现在收益和风险调整收益上，而不是所有指标都绝对压制 `FGSM`。

## 6. 与旧无约束多 seed 结果的对比

此前旧的无约束 `5%` 多随机种子实验中，主结果为：

- `partial_fgsm - partial_clean`
  - 年化超额收益均值：`-0.25310`
  - 最大回撤均值：`-0.04930`
  - `rank_ic_mean` 均值：`-0.01310`
  - 信息比率均值：`-1.77472`
- `partial_pgd - partial_clean`
  - 年化超额收益均值：`-0.36236`
  - 最大回撤均值：`-0.05440`
  - `rank_ic_mean` 均值：`-0.01451`
  - 信息比率均值：`-2.69830`

本轮 `physical_stat` 的对应结果则为：

- `partial_fgsm - partial_clean`
  - 年化超额收益均值：`-0.00007`
  - 最大回撤均值：`-0.01844`
  - `rank_ic_mean` 均值：`-0.01138`
  - 信息比率均值：`0.05959`
- `partial_pgd - partial_clean`
  - 年化超额收益均值：`-0.09202`
  - 最大回撤均值：`-0.03226`
  - `rank_ic_mean` 均值：`-0.01113`
  - 信息比率均值：`-0.61150`

这组对比说明：

1. 约束确实明显收紧了组合层攻击强度，特别是 `FGSM`；
2. 即便如此，`PGD` 的组合层退化仍然稳定成立；
3. 在 `rank_ic_mean` 上，受约束攻击依然保持了接近无约束版本的破坏水平，说明排序层面的扰动效果并未消失；
4. 因此，`physical_stat` 的代价不是“攻击失效”，而是“攻击更保守、收益侧退化幅度下降，但 PGD 仍稳定有效”。

## 7. 结论

本轮 `5%` 多随机种子组合层实验给出三个核心结论。

第一，`PGD + physical_stat` 在组合层面依然稳定成立。其对年化超额收益、最大回撤、`rank_ic_mean` 和信息比率四项主指标的退化方向，在五个随机种子上全部一致。

第二，`FGSM + physical_stat` 仍会稳定削弱排序能力和回撤表现，但其收益侧打击已经明显弱化，不再像无约束版本那样对年化超额收益和信息比率表现出稳定破坏。

第三，`physical_stat` 约束确实起到了“收紧攻击空间”的作用，但并没有消除 `PGD` 的组合层攻击效果。也就是说，在更贴近金融数据合理性的攻击定义下，组合退化证据仍然成立，只是其强度和指标分布发生了更细致的变化。

因此，从当前证据链看，后续组合层主攻击器应优先选择 `PGD + physical_stat`，而不是 `FGSM + physical_stat`。后者可以保留作为辅助对照，但不宜再作为组合层主要攻击结论的代表。
