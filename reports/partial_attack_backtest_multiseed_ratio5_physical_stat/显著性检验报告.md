# Physical-Stat 多随机种子显著性检验报告

## 1. 检验对象
- 目标目录：`partial_attack_backtest_multiseed_ratio5_physical_stat`
- 对照组：`partial_clean`
- 攻击组：`partial_fgsm`、`partial_pgd`

## 2. 主结果回顾
- `FGSM` 相对 `partial_clean` 的年化超额收益均值差值为 `-0.0001`，`RankIC` 均值差值为 `-0.011377`。
- `PGD` 相对 `partial_clean` 的年化超额收益均值差值为 `-0.0920`，`RankIC` 均值差值为 `-0.011131`。

## 3. 日度配对显著性
- `partial_fgsm_vs_partial_clean` 在 `超额收益` 上的日度平均差值为 `-0.000000`，`p=8.51e-01`。
- `partial_pgd_vs_partial_clean` 在 `超额收益` 上的日度平均差值为 `-0.000387`，`p=1.34e-02`。
- `partial_fgsm_vs_partial_clean` 在 `RankIC` 上的日度平均差值为 `-0.011377`，`p=3.77e-165`。
- `partial_pgd_vs_partial_clean` 在 `RankIC` 上的日度平均差值为 `-0.011131`，`p=3.90e-165`。

## 4. Block Bootstrap 区间
- `partial_fgsm_vs_partial_clean` 在 `年化超额收益` 上的 block bootstrap 区间为 `[-0.0693, 0.0690]`。
- `partial_pgd_vs_partial_clean` 在 `年化超额收益` 上的 block bootstrap 区间为 `[-0.1741, -0.0251]`。
- `partial_fgsm_vs_partial_clean` 在 `信息比率` 上的 block bootstrap 区间为 `[-0.3872, 0.5854]`。
- `partial_pgd_vs_partial_clean` 在 `信息比率` 上的 block bootstrap 区间为 `[-1.1613, -0.1059]`。
- `partial_fgsm_vs_partial_clean` 在 `最大回撤` 上的 block bootstrap 区间为 `[-0.0119, 0.0168]`。
- `partial_pgd_vs_partial_clean` 在 `最大回撤` 上的 block bootstrap 区间为 `[-0.0274, 0.0065]`。

## 5. 结论
- FGSM 在排序层面的扰动显著，但收益打击不具有统计显著性。
- PGD 在收益与排序两个层面都具有统计显著性，且年化收益退化区间稳定落在负向区间。
